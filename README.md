# Repozytorium z poragramownia objektowego
Pokazuje małe projekty które wykonałem w ramach zajęć laboratoryjnych w ramach tego przedmiotu
Główną atrakcją jest mała gra rpg wykonana na zaliczenie
Starałem się zwłaszcza w ostatnich projektach dbać o zasady SOLID.
Ponadto urzywałem CMaka jako systemu buildu.
W ramach projektu starałem się odzielić pliki źródłowe do folderu src a
pliki headerowe do osobnego folderu
