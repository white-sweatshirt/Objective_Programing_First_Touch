# Repozytorium z poragramownia objektowego
Pokazuje małe projekty które wykonałem w ramach zajęć laboratoryjnych w ramach tego przedmiotu<br>
Główną atrakcją jest mała gra rpg wykonana na zaliczenie<br>
Starałem się zwłaszcza w ostatnich projektach dbać o zasady SOLID.<br>
Ponadto urzywałem CMaka jako systemu buildu.<br>
W ramach projektu starałem się odzielić pliki źródłowe do folderu src a<br>
pliki headerowe do osobnego folderu<br>
